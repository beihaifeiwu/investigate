This archive contains 32 bit Windows DLL files required by SpatiaLite. The 
files were originally obtained from:

  http://www.gaia-gis.it/spatialite-2.3.1/binaries.html

The following packages/versions are included:

  * libiconv 1.9.2
  * proj 4.6.1
  * geos 3.1.1

In order for these libraries to be picked up by SpatiaLite they must be placed
in either the current working directory, or on the system DLL serach path such
as C:\WINDOWS\system32.
